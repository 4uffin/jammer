# signal jammer 
## play songs from cli

### usage
```
jammer [song]
```

### controls

_________________________
| key     | action      |
| `space` | play/pause  |
| `q`     | quit        |
| `→`     | forward 5s  |
| `←`     | backward 5s |
| `↑`     | volume up   |
| `↓`     | volume down |
| `m`     | mute/unmute |
| `L`     | toggle loop |
